# Sistema de Gestión de Librería 

## Descripción del Proyecto

El Sistema de Gestión de Librería es una aplicación web desarrollada para administrar de forma centralizada los procesos de una librería. El sistema permite gestionar libros, autores, editoriales, categorías, clientes, ventas, reservas e inventario, además de generar reportes y estadísticas para apoyar la toma de decisiones.

## Objetivo

Facilitar la administración de una librería mediante una plataforma web que permita controlar el inventario, registrar ventas, gestionar clientes y visualizar información estadística de manera rápida y eficiente.

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript ES6
- LocalStorage
- SessionStorage
- Chart.js
- GitHub
- Render

## Funcionalidades Principales

- Inicio de sesión con roles de usuario.
- Gestión de libros.
- Gestión de autores.
- Gestión de editoriales.
- Gestión de categorías.
- Gestión de clientes.
- Registro de ventas.
- Gestión de reservas.
- Control de inventario.
- Dashboard con indicadores y estadísticas.
- Reportes de ventas y clientes.

## Instrucciones de Despliegue

### Despliegue mediante GitHub

1. Crear un repositorio público en GitHub.
2. Subir todos los archivos del proyecto.
3. Verificar que el archivo `index.html` se encuentre en la raíz del repositorio.
4. Confirmar que los cambios se encuentren guardados mediante commits descriptivos.

### Despliegue mediante Render

1. Ingresar a Render.
2. Seleccionar la opción **New +**.
3. Elegir **Static Site**.
4. Conectar la cuenta de GitHub.
5. Seleccionar el repositorio del proyecto.
6. Configurar los siguientes parámetros:

- Branch: main
- Build Command: vacío
- Publish Directory: .

7. Presionar **Create Static Site**.
8. Esperar a que finalice el despliegue.
9. Verificar que el sitio se encuentre funcionando correctamente.
